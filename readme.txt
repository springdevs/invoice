== Description ==

Create, print & email PDF invoices & packing slips for WooCommerce orders.


## Features

- Automatic generate invoice for order
- Available to Download or Preview
- Download or Preview bulk invoices
- Automatically update invoice when order data is changed
- Easy to use !


== Installation ==

    = Installation from within WordPress =

        1. Visit 'Plugins > 'Add New'.
        2. Search for 'PDF Invoices & Packing Slips'.
        3. Install and activate the 'PDF Invoices & Packing Slips' plugin.

    = Manual installation =

        1. Upload the entire `PDF Invoices & Packing Slips` folder to the `/wp-content/plugins/` directory.
        2. Visit 'Plugins'.
        3. Activate the `PDF Invoices & Packing Slips` plugin.
